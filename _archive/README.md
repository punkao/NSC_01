## 🔍 บทเรียนจากการต้องมีโฟลเดอร์นี้

โฟลเดอร์นี้เกิดขึ้นเพราะระหว่างพัฒนา **เขียนทับไฟล์ผลลัพธ์เดิมโดยไม่เก็บสำเนา**
ทำให้ตอนเขียนรายงานหยิบตัวเลขจากเอกสารที่จดไว้คนละเวลา คนละคอนฟิก มาปนกัน
และไม่มีทางรู้ว่าตัวเลขไหนมาจากรอบไหน

**กติกาที่ใช้ต่อจากนี้**
1. ผลวัดทุกครั้งต้องเก็บ **ข้อมูลดิบรายเฟรม** ไม่ใช่เก็บแค่ผลสรุป
2. ตัวเลขในรายงานต้องมาจาก **`MEASUREMENTS.md` เท่านั้น** ซึ่งระบุคำสั่งทำซ้ำไว้ครบ
3. ไฟล์เก่าย้ายมาที่นี่ **พร้อมเหตุผลว่าทำไมไม่ใช้** — ไม่ลบทิ้ง เพื่อให้ตรวจสอบย้อนหลังได้
